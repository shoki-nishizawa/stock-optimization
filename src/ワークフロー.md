# 日本株ポートフォリオ最適化ツール ワークフロー図

以下のブロックはMermaid記法で記述されています。VS CodeなどのMarkdownプレビュー（右上のプレビューボタン）やGitHubなどで開くと、自動的に図として描画されます。

```mermaid
flowchart TD
    %% データ収集基盤
    subgraph DataPipeline ["1. 裏側のデータ収集（自動で動く部分）"]
        direction TB
        F_API[Yahoo Finance\nPER・ROEなど] -->|月1回 時間をかけて取得\nfetch_fundamentals_local.py| F_CSV[(fundamentals.csv\n財務データ)]
        
        JPX[JPX 銘柄マスタ公式] -->|毎日更新| U_DB
        P_API[Yahoo Finance\n毎日の株価] -->|毎日更新| U_DB[update_database.py\nデータを一つにまとめる]
        F_CSV -->|合体させる| U_DB
        
        U_DB -->|完成！| FULL_DB[("stock_database.csv\n(このツール専用のデータベース)")]
    end

    %% アプリ側
    subgraph AppProcess ["2. アプリでの動き（ユーザーが画面を操作する部分）"]
        direction TB
        USER(["ユーザー\n予算・業種・ボラ上限を入力"]) --> UI_INPUT
        
        FULL_DB -->|一瞬で読み込む| DL(data_loader.py)
        
        UI_INPUT[条件を受け取る] --> DL
        
        DL -->|①割安かスコアで評価\n②値動きが激しすぎる銘柄を除外| CANDIDATES[優良な候補銘柄を絞り込む\n(第一段階クリア)]
        
        CANDIDATES -->|グラフ用に株価を取得| VIS(visualizer.py)
        CANDIDATES -->|候補リストを渡す| OPT(optimizer.py)
        UI_INPUT -->|予算や集中投資制限| OPT
        
        OPT -->|AI最適化！(OR-Tools)\n予算内で一番儲かる株数の組み合わせを計算\n(第二段階クリア)| RESULT[最終的なおすすめ\nポートフォリオ]
        VIS -->|株価グラフ作成| RESULT
        
    end

    RESULT --> FINAL_VIEW([Webダッシュボードに表示])
    
    %% 色付け
    classDef file class:#e3f2fd,stroke:#1e88e5,stroke-width:2px,color:#000
    class F_CSV,FULL_DB file
```
