# 日本株 ポートフォリオ最適化ツール

本ツールは、「予算」や「業種」「値動きのプレファレンス」に基づいて、日本株（JPX公式データ）の中から効率の良いポートフォリオ（何株買うべきか）と期待利益をGoogle OR-Tools（ナップサック問題）を用いて数学的に算出するツールです。

また、選出された投資先候補銘柄の過去1年間の株価推移グラフ(`.png`画像)を自動生成します。

## 使い方

1. `src/` フォルダ配下にある `config.json` の内容を編集し、抽出条件を設定します。
2. コマンドラインからメインプログラムを実行します。
   ```bash
   # 仮想環境と必要なライブラリがインストールされた状態で実行
   python src/main.py
   ```
3. 計算結果がコンソールに出力され、同じフォルダに株価推移グラフの画像（`candidate_trends.png`）が保存されます。

---

## 設定ファイル (`config.json`) の仕様

`config.json` では、以下の項目を自由に設定してスクリーニングと最適化の方針を決定できます。

### 基本設定
- **`budget`** (整数)
  - 投資に回す総予算（円）です。プログラムはこの予算内で最大の利益が見込める組み合わせ（1単元＝100株単位）を計算します。
  - 例: `2000000` （200万円）

### フィルタリング設定 (`filter`)
JPX（日本取引所グループ）の全上場銘柄マスターから、大枠の条件で銘柄を絞り込みます。
※空のリスト `[]` を指定すると、フィルタリングされず全て対象になります。

- **`sector`** (文字列のリスト)
  - ターゲットとする「33業種区分」を指定します。
  - 主な指定可能文字列: `"医薬品"`, `"情報・通信業"`, `"電気機器"`, `"銀行業"`, `"サービス業"`, `"輸送用機器"`, `"小売業"` など
  - 例: `["医薬品", "情報・通信業"]`

- **`market`** (文字列のリスト)
  - ターゲットとする「市場・商品区分」を指定します。
  - 主な指定可能文字列: `"プライム（内国株式）"`, `"スタンダード（内国株式）"`, `"グロース（内国株式）"`
  - 例: `["プライム（内国株式）"]`

### 抽出戦略・ソート設定 (`strategy`)
フィルタリングされた数十〜数百の銘柄から、過去1年分のデータ（yfinance経由）を計算し、「どの銘柄を上位に据えて最適化するか」を決定します。

- **`sort_by`** (文字列)
  - ランキングの基準（ソート条件）を指定します。
  - 指定可能な値:
    - `"volatility_high"`: ボラティリティ（日次株価変動率の標準偏差）が大きい順＝**「値動きの激しさ」**を重視します。「上がるときは一気に上がる銘柄」を探す実験に向いています。
    - `"return_high"`: 過去1年間の単純な上昇率（リターン）が大きい順＝**「直近1年で一番実績良かった銘柄」**を重視します。
    
- **`top_n`** (整数)
  - `sort_by` でランキングされた上位から、具体的に「何社」を最適化ソルバーに渡してポートフォリオを組むかを指定します。
  - （ここであまり大きい数字を指定しすぎると処理が重くなります）
  - 例: `10` （上位10社を候補とする）

---

### config.json の記述例（ボラティリティ重視の医療系トップ10）
```json
{
  "budget": 2000000,
  "filter": {
    "sector": ["医薬品"],
    "market": ["プライム（内国株式）"]
  },
  "strategy": {
    "sort_by": "volatility_high",
    "top_n": 10
  }
}
```
